# Student Dropout Prediction using BPNN

## Course
ISP560 – Machine Learning

## Dataset
UCI Machine Learning Repository – Student Dropout Dataset

## Objective
To build a classification model using a Backpropagation Neural Network (BPNN) to predict student dropout.

## Steps
1. Data preprocessing (missing values, encoding, normalization)
2. Train-test split
3. BPNN model training using TensorFlow/Keras
4. Model evaluation (accuracy, confusion matrix)

## Technologies
- Python
- NumPy
- Pandas
- Scikit-learn
- TensorFlow / Keras
